## Install

**1. If you have installed Git**

open `/Applications/Utilities/Terminal.app`ï¼Œrun command

`cd ~/Downloads && rm -rf WeChatPlugin-MacOS && git clone https://github.com/TKkk-iOSer/WeChatPlugin-MacOS.git --depth=1 && ./WeChatPlugin-MacOS/Other/Install.sh`, and enter the mac password.

**2. Normal install**

* click `clone or download` button to download project and unzip itï¼Œopen Terminal.appï¼ŒDrag the `Install.sh` file(in `Other` Folder) to Terminal.
